## Description
We present a collaborative **VR escape room** where each player has a different power needed to solve a series of puzzles.

We developed this project using **Unity** and **Ubiq** for avatar movements and synchronized object interactions. 
We aim to create a positive experience for the players in terms of immersion, presence, social presence and cognitive load. The puzzles were designed to maximise the collaboration between the users, allowing each player to contribute to the solution using their unique power.
### Challenges
The challenges encountered while developing this kind of experience were both technical, such as multiplayer synchronization, and related to the game design, specifically puzzle creation and super-powers design. 
### Evaluation
We did a brief evaluation of the demo with external participants testing cognitive load, through the NASA Task Load Index (Hart and Staveland, 1988), and usability, with SUS.
While the initial results are promising, more large scale and in-depth testing is required, including interviews and objective performance tests of the system.

2. Go to master branch
3. Open Project in Unity
4. Connect headset to PC
5. Build & Run EscapeRoom scene (Found under the 'Scenes' folder)
